# Web Accessibility (a11y) & WCAG
- **Marcado:** Usa elementos HTML semánticos nativos (ej. `<button>` en lugar de `<div onClick>`).
- **Navegación:** Diseña focus rings visibles y asegura una tabulación lógica en toda la página.
- **ARIA:** Usa atributos ARIA (`aria-live`, `aria-expanded`, etc.) solo cuando los nativos no cubran la interacción.
