# Type-Safe GraphQL with Codegen
- **Generación:** Corre siempre el generador de tipos después de alterar schemas o consultas para evitar discrepancias de tipos.
- **Fragmentos:** Agrupa consultas repetidas en fragmentos reutilizables en el cliente para estructurar mejor el código y el cache.
