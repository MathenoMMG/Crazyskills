# Agentic Tool Use & MCP Engineering
- **Manejo de Errores:** Diseña llamadas de herramientas con control robusto y reintentos automáticos elegantes en caso de fallos de red.
- **Context Economy:** Envía únicamente la información estrictamente necesaria a las APIs del agente para no saturar el contexto.
- **JSON Schemas:** Define esquemas estrictos de entrada/salida para asegurar llamadas consistentes sin errores de tipos.
