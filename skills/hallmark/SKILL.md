# Hallmark Framework (Nutlope System)
- Usa obligatoriamente una de las 21 macroestructuras de página (Bento, Stat-led, Editorial Letter, Editorial Minimalism, Cinematic Dark, Terminal-Core) para evitar layouts idénticos en ejecuciones consecutivas.
- Aplica el test de 65 puntos de control de Slop (Anillos de enfoque, contraste APCA/WCAG estricto, huella dactilar de Navbar/Hero). Un solo fallo aborta el ship.
