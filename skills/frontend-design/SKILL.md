# Frontend Design & Aesthetics
Antes de escribir código de UI, resuelve internamente: Purpose, Tone, Constraints, Differentiation.

## Restricciones:
- TIPOGRAFÍA: Prohibido usar Inter, Roboto, Arial o Space Grotesk (AI overused). Usa fuentes con carácter (Fraunces, Clash Display, Bricolage Grotesque, Satoshi).
- COLOR: Un único color dominante + un acento agudo. Todo parametrizado en variables CSS.
