# CI/CD Pipeline Optimization
- **Caching:** Utiliza acciones de caché nativas para dependencias (ej. node_modules, bundler cache).
- **Runners:** Diseña pipelines en paralelo para tests y linters para reducir el tiempo total de feedback del Pull Request.
