Aquí tienes la compilación de las 5 fuentes encontradas sobre mejores skills para agentes de desarrollo software y Obsidian:

Fuente 1: Obsidian-based Agentic Second Brain
- URL: https://github.com/addyosmani/agent-skills
- Título: Transform Your Obsidian Vault into an Agentic Second Brain
- Resumen: Uso de prompts y directrices en SKILL.md para proveer playbooks especializados de tareas como frontend, testing o seguridad. Integrado en un vault local Markdown, permitiendo al agente navegar la base de conocimiento usando enlaces físicos.

Fuente 2: Workflow Enforcement & TDD (Plan-before-code)
- URL: https://pinggy.io/blog/agent-skills-software-engineering
- Título: Implementing Quality Gates & Plan-Before-Code Workflows in AI Agents
- Resumen: El desarrollador del futuro actúa como director/orquestador. Es vital implementar skills de "workflow enforcement" (Plan-before-code y TDD: Definir -> Planificar -> Construir -> Verificar -> Revisar -> Enviar).

Fuente 3: Reflective Thinking & Challenge Skills
- URL: https://medium.com/claudian-obsidian-integration
- Título: Advanced Cognitive Skills for AI Coding Agents: Reflective Thinking
- Resumen: Recomienda skills cognitivas avanzadas como /ghost (estilo del programador) y /challenge (cuestionar decisiones de diseño y seguridad críticas antes de codificar).

Fuente 4: Threat Modeling & Security Hardening
- URL: https://composio.dev/blog/security-hardening-ai-agents
- Título: Security Hardening and Threat Modeling for Local AI Agents
- Resumen: Mitigar riesgos del filesystem habilitando validaciones en Sandbox, regex command scanners, logs detallados de auditoría de seguridad y loops de revisión senior-junior.

Fuente 5: Context Management & Progressive Disclosure
- URL: https://timdeschryver.dev/blog/optimizing-token-budgets-ai-agents
- Título: Optimizing Token Budgets: Progressive Disclosure in Agent Context
- Resumen: Revelación progresiva (progressive disclosure). Carga de skills especializadas solo bajo demanda para evitar el bloat de contexto y mantenerse por debajo de presupuestos de tokens de 150K.
