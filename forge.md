---
description: Lanza obsidian-forge para revisar observaciones en Engram y evolucionar o crear skills. Uso: /forge [nombre-skill] para una skill específica, /forge --all para review completo de todas las skills con 3+ observaciones pendientes.
---

Activa obsidian-forge con el argumento $ARGUMENTS.
Si --all: engram search scope:skill:* para detectar todos los scopes con 3+ obs abiertas, procésalos.
Si nombre: procesa solo ese scope.
Si nueva:[nombre]: lanza flujo de creación de skill nueva.
